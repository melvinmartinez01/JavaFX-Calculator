# MelvinMartinez - JavaFX Calculator

A JavaFX calculator application built for CS 55 at Santa Monica College.

## Features
- Add, subtract, multiply, and divide two integers
- Error handling for invalid input
- Built with JavaFX and FXML

## Files
- MelvinMartinezApplication.java
- MelvinMartinezController.java
- melvinmartinez.fxml
